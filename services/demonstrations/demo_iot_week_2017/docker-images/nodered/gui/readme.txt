How to configure node-RED for new look
----------------------------------------

This folder contains the new css file, new images and new setting.js file used to change the look of  Node-RED.

1) Copy this folder in some location of your hard drive
2) Open setting.js file present in this folder, and change the path of the image, css and favicon with your absolute path (where you extract these folders, for instance)
3) Replace  installation setting.js file with new setting.js file 
4) Restart Node-RED server
